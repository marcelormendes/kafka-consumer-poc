# Template NodeJS with Express, Typescript, Docker and Kafka

## How to Run the Project

You need to install on your machine:

-   docker
-   nodejs version 12.x
-   yarn

Run the following commands

```
docker-compose up -d   # if you use docker
yarn                   # install node_modules
cp .env.example .env   # copy the environment example configs
yarn dev               # start the server
```
